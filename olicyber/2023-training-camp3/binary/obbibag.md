## Overview

The provided ELF binary displays an ASCII animation composed of alternating frames.

## Solve

Running the binary:

```bash
./obbibag
```

shows two alternating frames, one containing the gabibbo and one containing the flag.

Interrupting the execution (Ctrl+C) while the flag frame is displayed allows the flag to be read  and copied directly from the terminal output.
