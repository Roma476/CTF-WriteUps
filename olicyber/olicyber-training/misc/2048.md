## Overview

The challenge asks the user to solve 2048 arithmetic operations:

```
Benvenuti a 2048
Rispondi inserendo il risultato delle prossime 2048 operazioni:
DIVISIONE_INTERA 25 51
```

Manually solving all operations is impractical, making automation the intended solution.

## Solve

A simple script can parse each operation, compute the result, and send it back to the server:

```python
from pwn import *

chall = remote("2048.challs.olicyber.it", 10007)

chall.recvuntil(b"operazioni:")

for i in range(2048):
    data = chall.recv().decode().split()

    if not data:
        continue

    op = data[0]
    n1 = int(data[1])
    n2 = int(data[2])

    if op == "POTENZA":
        res = pow(n1, n2)
    elif op == "DIVISIONE_INTERA":
        res = n1 // n2
    elif op == "PRODOTTO":
        res = n1 * n2
    elif op == "SOMMA":
        res = n1 + n2
    elif op == "DIFFERENZA":
        res = n1 - n2

    chall.sendline(str(res).encode())

chall.interactive()
```

After correctly answering all 2048 operations, the service returns the flag.
